# vincentsassard.github.io

Personal academic website of **Vincent Sassard**, Ph.D. Candidate in seismology at Purdue University.

🌐 **Live site:** [vincentsassard.github.io](https://vincentsassard.github.io)

---

## About

This site presents my research, publications, presentations, teaching, and outreach activities. It is built with plain HTML, CSS, and JavaScript — no frameworks or dependencies — and hosted for free via GitHub Pages.

## Structure

```
├── index.html           # About / home
├── research.html        # Research projects & experience
├── presentations.html   # Publications & conference presentations
├── teaching.html        # Teaching, outreach & service
├── photos.html          # Field work & expedition photos
├── contact.html         # Contact information
├── style.css            # Shared stylesheet
├── script.js            # Navigation & scroll animations
├── sitemap.xml          # For Google Search indexing
├── images/              # Photos, figures, favicon
└── Vincent_SASSARD_CV_2026-05-25.pdf
```

## Updating the site

Edit any `.html` file directly on GitHub and commit — the site redeploys automatically within ~60 seconds.

**Common updates:**
- **New publication** → add a `pub-card` block in `presentations.html`
- **New presentation** → add a `talk-item` block in `presentations.html`
- **New photo** → upload to `images/` and add a `gallery-item` block in `photos.html`
- **New CV** → upload the PDF and update the filename in all `href` links
